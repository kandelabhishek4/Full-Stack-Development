Bootstrap Framework

Bootstrap provides responsive grid classes and ready-made UI components for faster development.

Overview

Bootstrap is a popular open-source front-end framework for building responsive, mobile-first websites quickly. Originally developed by Twitter, it now powers a huge share of modern web projects with ready-made CSS classes, components, and JavaScript plugins. Rather than writing layout and component styling from scratch for every project, developers can rely on Bootstrap's pre-tested grid system and UI elements, which significantly speeds up development while still producing pages that look consistent and professional across different browsers and screen sizes. It is especially useful for prototyping, smaller projects, or teams that need to move fast without sacrificing responsiveness or accessibility. Because Bootstrap is built on plain HTML, CSS, and JavaScript, the skills learned while using it transfer directly to working with custom-built frameworks and design systems later on.

Responsive Grid System

Bootstrap's best-known feature is its 12-column grid, which divides the page into equal columns so layouts can adapt to any screen size.

  • col-sm- for small screens such as large phones, keeping content readable once the viewport gets a little wider than a typical phone in portrait mode.
  • col-md- for tablets and medium viewports, where there is enough room to show two or three columns side by side.
  • col-lg- and col-xl- for laptops and desktops, allowing layouts to spread across more columns once there is ample horizontal space.

Mobile-First & Utility Classes

Styles are optimised for small screens first, then scaled up, which improves performance and usability across devices.

  • Utility classes for spacing, colour, and typography, such as margin and padding helpers, so common adjustments can be made directly in the HTML without writing extra CSS.
  • Alignment and display helpers without custom CSS, making it quick to centre, hide, or rearrange elements just by adding a class.
  • Consistent design tokens shared across components, which keeps colours, spacing, and sizing uniform throughout the whole site.

Pre-Built Components

A wide library of ready components saves development time and keeps designs consistent.

  • Navigation bars, buttons, and cards, which arrive already styled and responsive, so they only need content rather than layout work from scratch.
  • Forms, modals, and alerts, each built with consistent spacing and styling so they fit visually with the rest of a Bootstrap-based page.
  • Carousels and dropdown menus, which would normally require custom JavaScript but come ready to use with just a few HTML classes and attributes.

JavaScript Interactivity

Built-in plugins add interactivity using simple HTML data attributes instead of custom scripts.

  • Modals and collapsible menus, which can open, close, and animate without the developer writing any JavaScript themselves.
  • Tooltips and dropdowns, which appear and position themselves automatically near the element that triggered them.
  • Powered by data-bs-toggle and data-bs-target attributes, which tell Bootstrap's JavaScript which component to control and which element should respond to it.

Compatibility & Considerations

Bootstrap works consistently across Chrome, Firefox, Edge, and Safari, backed by strong community support and documentation.

Used without customisation, Bootstrap sites can look similar to one another — but applying custom CSS on top of the framework keeps designs unique while still benefiting from its speed and consistency.
