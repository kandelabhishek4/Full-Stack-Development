CSS

CSS controls colours, typography, spacing, layout, and responsive design.

Overview

CSS (Cascading Style Sheets) controls the presentation and visual design of HTML documents. While HTML provides structure, CSS decides how that content looks — layout, colours, fonts, spacing, and responsiveness — keeping content and design cleanly separated. This separation means the same HTML can be restyled completely just by changing the CSS, without ever touching the underlying content, which is part of what makes large websites manageable to update and maintain over time. CSS also follows a cascade, where rules can come from the browser, the website, and even the user, with specificity and order deciding which rule ultimately wins when several apply to the same element. From simple colour and spacing changes to fully responsive layouts that adapt across phones, tablets, and desktops, CSS is what turns a plain structured document into a polished, professional-looking interface.

How CSS Works

A CSS rule pairs a selector with a declaration block: the selector targets HTML elements, and the declaration sets their style properties.

  • Selectors can target tags, classes, or unique IDs, giving control over exactly which elements a style applies to, from every paragraph on the page down to one specific button.
  • Declarations define property and value pairs, such as color: blue;, where the property names what is being changed and the value sets how it should look.
  • Rules can be combined for structured, efficient styling, so multiple selectors can share the same declarations instead of repeating the same styles over and over.

The Box Model

Every HTML element is treated as a rectangular box, which is essential for controlling spacing and layout.

  • Content — the actual data inside the element, such as text or an image, which is the innermost part of the box.
  • Padding — space inside the element, around the content, used to stop content from feeling cramped against its own edges.
  • Border — the line surrounding the padding, which can be styled with different widths, colours, and styles to frame the element.
  • Margin — space outside the element, separating it from others, which controls how far apart elements sit from their neighbours.

Flexbox & CSS Grid

Modern layout relies on two complementary systems.

  • Flexbox — one-dimensional layouts; ideal for navbars and centering, since it arranges items neatly along a single row or column and adjusts spacing automatically.
  • CSS Grid — two-dimensional layouts with rows and columns, making it well suited to page layouts, image galleries, and dashboards that need precise alignment in both directions.

Responsive Design & Animation

Media queries let a website adapt its styles to different screen widths, improving usability across devices.

  • Media queries adjust layout for desktop, tablet, and mobile, so the same page can rearrange its columns, font sizes, or spacing depending on the width of the screen viewing it.
  • Transitions create smooth hover and state changes, making interactions like button hovers or menu openings feel gradual rather than instant and jarring.
  • Animations add movement and visual engagement, which can guide attention or simply make a page feel more polished and alive.
CSS works alongside HTML and JavaScript to create modern, responsive, user-friendly websites — a strong grasp of it is essential for professional front-end development.
