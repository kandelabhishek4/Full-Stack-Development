JavaScript

JavaScript adds interaction, logic, storage, and dynamic content to web pages.

JS

Overview

JavaScript is a high-level, interpreted programming language that adds interactive and dynamic behaviour to websites. Alongside HTML and CSS, it's one of the three core web technologies — handling logic and responding to user actions. Unlike HTML and CSS, which describe structure and appearance, JavaScript is a full programming language capable of making decisions, storing and processing data, and reacting to events as they happen, which is what allows a page to update itself instantly rather than reloading for every change. It runs directly in the browser on the frontend, but the same language, through environments like Node.js, can also power backend servers, making it one of the few technologies that can be used across the entire stack of a web application. This versatility, combined with a huge ecosystem of libraries and frameworks, is why JavaScript remains one of the most widely used programming languages in the world.

The DOM

On the frontend, JavaScript runs in the browser and manipulates the Document Object Model (DOM).

  • The DOM represents an HTML document as a tree of objects, with each element on the page available to JavaScript as something it can read or change.
  • JavaScript can change text, styles, and layout without reloading the page, which is what makes interfaces feel instant rather than waiting for a fresh page load each time.
  • Essential for forms, dashboards, and single-page applications, where content needs to update continuously in response to what the user is doing.

Variables & Functions

JavaScript supports multiple data types and reusable blocks of logic.

  • Data types: strings, numbers, booleans, arrays, objects, null, undefined, each suited to representing a different kind of value, from simple text to entire collections of information.
  • let and const are preferred over var for better scope control, helping avoid bugs caused by variables leaking outside the block they were meant to stay in.
  • Functions keep code organised and reusable, letting the same block of logic run multiple times from different parts of a program instead of being duplicated.

Events & Modern Features

Event listeners let JavaScript respond to user actions like clicks, typing, and form submissions.

  • Arrow functions, template literals, and destructuring, modern syntax features that make JavaScript code shorter and easier to read.
  • Spread operators, classes, and modules, which help organise larger codebases and make copying or combining data more straightforward.
  • Promises and async/await for asynchronous code, allowing JavaScript to wait for things like server responses without freezing the rest of the page.

Fetch API & Local Storage

JavaScript can communicate with external servers and persist data in the browser.

  • Fetch API retrieves data, typically in JSON format, sending a request to a server and handling the response once it arrives.
  • Used for live data such as weather, users, or products, keeping a page up to date with information that changes outside of the website itself.
  • Local Storage saves preferences or tasks even after the browser closes, so information the user entered is still there the next time they visit.
JavaScript plays a critical role in both frontend and backend development, making it essential for becoming a Full Stack Developer.
